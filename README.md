# Master Infuser

> **MASTER INFUSER IS CURRENTLY IN ALPHA. CRASHES AND BUGS MAY ARISE.**
>
> *Please report any issues on [GitHub](https://github.com/Funoro/MasterInfuser/issues).*

Master Infuser adds the machine **Auto-Infuser** which aims to simplify the automation of crafting tiered essences in Mystical Agriculture.

Instead of relying on the player to provide an infusion crystal, the Auto-infuser takes these crystals in its recipe, allowing the machine to function with a single input and output slot. This makes automation with mods like AE2 incredibly easy.

The machine integrates seamlessly with mods that use the built-in item and energy interfaces provided by NeoForge (e.g. Applied Energistics 2, Powah! and more). It also supports the machine upgrades provided by Mystical Agriculture.

### Acknowledgement

Although BlakeBr0's Mystical Automation mod adds an Infuser block which achieves similar functionality, the simple automation of that machine is not possible as is. This is due to how it handles the infusion crystal, and because the player has to manually select the output essence.
